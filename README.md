# java-cli-gradle-findbugs-jacoco-jbehave-testng-hello-world

## Description
A POC app using testng,
findbugs, jacoco, jbehave framework.

## Tech stack
- java
- gradle
  - testng  
  - findbugs
  - jacoco
  - jbehave

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
